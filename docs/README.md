# Documentation

This folder contains cleaned extracts, a working summary, reusable graphics, and the original source files.

## Start Here

- `working-brief.md` is the best single-file summary of the organization, mission, model, priorities, and open questions.
- `proposal-extract.md` is the cleaned text extract from the proposal document.
- `slides-extract.md` is the cleaned text extract from the slide deck, including notes about visual-only slides and embedded diagram graphics.
- `assets/README.md` catalogs the extracted artwork and icon files.
